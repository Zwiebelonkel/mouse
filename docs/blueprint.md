# **App Name**: Mouse Milker

## Core Features:

- Clickable Mouse: Display a mouse image that the user can click.
- Click Counter: Keep track of the number of clicks.
- Milking Result: Displays the milking result when the user has clicked 10 times.
- Start Message: When starting the game the message should be please milk the mouse

## Style Guidelines:

- Primary color: Light blue (#ADD8E6), reminiscent of milk and serenity.
- Background color: Very light blue (#F0F8FF), almost white to maintain a clean look.
- Accent color: Pale yellow (#FAFAD2) for the 'milked' message.
- Body and headline font: 'Inter' for a modern, clean interface.
- Use a simple mouse icon, possibly cartoonish, for the clickable image.
- Subtle animation on the mouse when clicked, such as a slight bounce or shake.
- Center the mouse on the screen, with the click counter displayed clearly.
